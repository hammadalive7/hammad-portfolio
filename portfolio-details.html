Community Chat App

Group Tea is a cross-platform community chatting app developed using Flutter. It enables users to chat in groups and exchange messages, images and videos with each other. Firebase Authentication was used for user registration and Firebase Firestore was utilized as the backend database to store user and group details. The app is designed with an intuitive user interface and supports real-time messaging. Group Tea is a personal project that was completed in August 2022, showcasing the developer's proficiency in Flutter and mobile app development.
